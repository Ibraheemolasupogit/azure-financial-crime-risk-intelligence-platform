# Data Privacy And Synthetic Data

This project is designed for synthetic and sample data only.

The repository must not contain:

- Real customer records
- Real account numbers
- Real card numbers
- Real banking transactions
- Personal data
- Confidential financial institution data

Future milestones will generate fictional customers, accounts, transactions, alerts, and investigation artifacts for local testing and portfolio demonstration.

Any future examples that resemble regulated financial workflows must remain synthetic and clearly labeled.
